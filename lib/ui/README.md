# lib/ui/

Design system et composants UI transverses réutilisables.

## Sous-dossiers

- `theme/` — Tokens de design, typographie, `theme_v2.dart` (à venir)
- `components/` — Composants atomiques : `EabButton`, `EabField`, `EabTable`, `EabDialog`, états (Empty/Error/Skeleton)
- `layout/` — Mise en page : `AppShell`, `PageHeader`, spacing, responsive helpers
