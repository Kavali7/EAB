# lib/ui/components/

Composants atomiques réutilisables du design system EAB.

## Composants prévus

- `eab_button.dart` — Bouton standard avec variantes (primary, secondary, danger)
- `eab_text_field.dart` — Champ texte avec validation intégrée
- `eab_select_field.dart` — Sélecteur dropdown
- `eab_date_field.dart` — Sélecteur de date
- `eab_table.dart` — Tableau paginé responsive (remplace PaginatedDataTable)
- `eab_dialog.dart` — Dialog modal standard
- `eab_card.dart` — Carte avec variantes
- `empty_state.dart` — État vide standard
- `error_state.dart` — État d'erreur standard
- `skeleton_loader.dart` — Chargement par squelette
