## 🔐 Connexion

### Première connexion
1. Accédez à l'application EAB
2. Cliquez sur "Créer un compte"
3. Renseignez votre email et mot de passe
4. Confirmez votre email via le lien reçu

### Connexion habituelle
1. Entrez votre email
2. Entrez votre mot de passe
3. Cliquez sur "Se connecter"

## 👥 Gestion des membres

### Ajouter un membre
1. Accédez à **Membres** dans le menu
2. Cliquez sur **+ Nouveau membre**
3. Remplissez les informations :
   - Identité (nom, prénom, date de naissance)
   - Contact (téléphone, email, adresse)
   - Vie spirituelle (conversion, baptême, main d'association)
   - Statut (rôle dans l'église)
4. Cliquez sur **Enregistrer**

### Modifier un membre
1. Recherchez le membre dans la liste
2. Cliquez sur son nom
3. Modifiez les informations souhaitées
4. Cliquez sur **Enregistrer**

### Statuts possibles
| Statut | Description |
|--------|-------------|
| Actif | Membre régulier |
| Inactif | Membre absent depuis longtemps |
| Transféré | Parti vers une autre assemblée |
| Parti | Quitté l'église |
| Décédé | Membre décédé |

### Rôles dans l'église
- Membre
- Pasteur
- Ancien
- Diacre / Diaconesse
- Évangéliste
- Autre officier

## 📅 Programmes et activités

### Types d'activités
- **Culte** : Service dominical
- **Évangélisation de masse** : Croisade publique
- **Évangélisation porte-à-porte** : Visite aux maisons
- **Baptêmes** : Cérémonie de baptême
- **Main d'association** : Intégration de nouveaux membres
- **Sainte Cène** : Communion
- **Réunion de prière** : Prière collective
- **Mariage** : Cérémonie de mariage
- **École du dimanche** : Formation biblique
- **Visite** : Visite pastorale ou officielle

### Enregistrer une activité
1. Accédez à **Programmes** dans le menu
2. Cliquez sur **+ Nouvelle activité**
3. Sélectionnez le type
4. Renseignez la date et le lieu
5. Entrez les statistiques de fréquentation :
   - Hommes / Femmes / Garçons / Filles
   - Nouvelles conversions
6. Cliquez sur **Enregistrer**

## 💰 Comptabilité

### Plan comptable SYSCEBNL

Le plan comptable est organisé selon le SYSCEBNL :

| Classe | Nature | Exemples |
|--------|--------|----------|
| 2 | Immobilisations | Terrains, bâtiments, mobilier |
| 4 | Tiers | Fournisseurs, membres |
| 5 | Trésorerie | Caisse, banque |
| 6 | Charges | Loyers, transports, aides |
| 7 | Produits | Dîmes, offrandes, dons |

### Journaux comptables
- **CAI** : Journal de caisse (espèces)
- **BAN** : Journal de banque (virements, chèques)
- **OD** : Opérations diverses

### Saisir une écriture comptable

1. Accédez à **Comptabilité** > **Saisie**
2. Sélectionnez le journal
3. Entrez la date et le libellé
4. Ajoutez les lignes :
   - Compte à débiter
   - Compte à créditer
   - Montant
5. Vérifiez que l'écriture est équilibrée (Total Débit = Total Crédit)
6. Cliquez sur **Enregistrer** (brouillon) ou **Valider**

### Statuts des écritures
| Statut | Description | Modifiable ? |
|--------|-------------|--------------|
| Brouillon | En cours de saisie | ✅ Oui |
| Validée | Approuvée | ❌ Non |
| Clôturée | Exercice fermé | ❌ Non |

### Centres analytiques

Les centres analytiques permettent de suivre les coûts par :
- Assemblée locale
- Projet spécifique
- Département

## 📊 Rapports

### Rapport mensuel EAB

Le rapport mensuel consolide les données de chaque assemblée :

**Section Membres :**
- Effectifs actifs (hommes, femmes)
- Nouveaux convertis du mois
- Nouveaux baptisés
- Décès

**Section Activités :**
- Nombre de cultes
- Évangélisations réalisées
- Baptêmes effectués
- Mariages célébrés

**Section Finances :**
- Total des produits (recettes)
- Total des charges (dépenses)
- Résultat du mois

### Générer un rapport
1. Accédez à **Rapports** > **Rapport mensuel**
2. Sélectionnez l'assemblée
3. Choisissez l'année et le mois
4. Cliquez sur **Générer**
5. Visualisez ou exportez le rapport
